Archidekt To JSON Converter

Intro

This is the tool I use to set up the back end for tools on this site. It does two things:

  1. Parses a deck list exported from Archidekt and creates a JSON object from it that's easier to work with.
  2. Create lines for a bash script to pull down images from scryfall to use so I'm not hitting their servers for my tools.

I built these tools for my stuff. There's no reason they couldn't be used for other tools as well.
